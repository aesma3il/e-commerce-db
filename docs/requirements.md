# E-Commerce Database Requirements

## 1. Product Catalog Management
The system should store product information including:
- Name
- Description
- Price
- Stock Quantity
- Images

---

## 2. Customer Information
The system should store customer data including:
- Full Name
- Email
- Password
- Contact Details
- Shipping Address

---

## 3. Order Information
The system should store orders including:
- Order Number
- Customer
- Order Date
- Products
- Quantities
- Prices
- Status

### Order Status
- Pending
- Processing
- Shipped
- Delivered
- Cancelled
- Refunded

---

## 4. Payment Transactions
The system should store payment data:
- Transaction ID
- Customer
- Amount
- Payment Method
- Timestamp

---

## 5. Shipping
The system should store shipping data:
- Order ID
- Carrier
- Tracking Number
- Status
- Estimated Delivery
- Actual Delivery

### Shipping Status
- Processing
- Out for Delivery
- Delivered
- Return to Sender
- On Hold
- Delayed
- Lost

---

## 6. Reviews and Ratings
The system should store reviews:
- Product ID
- Customer ID
- Rating (1–5)
- Review Text
- Timestamp
